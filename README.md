🚀 Инструкция по развертыванию
Установите Node.js и MySQL.
Создайте папку проекта и скопируйте все вышеуказанные файлы в соответствующие подпапки.
Выполните npm install в корне проекта.
Создайте базу данных и таблицы, выполнив sql/init.sql в MySQL. Не забудьте заменить хеш пароля для admin.
Настройте подключение к БД в файле .env.
Установите и запустите recorder_agent.py на машине которую будем слушать.
Заполните config/agents.json реальными IP-адресами ваших агентов (где запущен recorder_agent.py).
Запустите сервер: npm start.
Откройте браузер на http://localhost:8000 и войдите под admin / admin123
